# El Padrino — Restaurant & Cigar Lounge

Site vitrine du restaurant de luxe et cigar lounge **El Padrino** (Abidjan, Plateau).

Page unique immersive : hero vidéo, carte interactive du Chef Charbel Akiki,
cigar lounge, galerie, réservation via WhatsApp et localisation.

Statique — aucune dépendance de build. Ouvrir `index.html` ou servir le dossier.
